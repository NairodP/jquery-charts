Types de Graphiques

jQuery-Charts offre une variété de types de graphiques adaptés à différents besoins de visualisation. La bibliothèque maintient une syntaxe uniforme tout en permettant un rendu spécifique pour chaque type de graphique.

Compatibilité et utilisations recommandées

Graphiques circulaires

  • pie - Répartition de valeurs dans un ensemble complet
  • donut - Comme pie mais avec espace central pour informations supplémentaires
  • polar - Répartition circulaire avec valeurs radiales
  • radar - Comparaison multi-variables sur axes radiaux
  • radialBar - Progression circulaire par catégorie
↔ Les types pie et donut sont interchangeables (même données, avec/sans trou central)
↔ Les types polar et radar sont interchangeables (même données, visualisation différente)

Graphiques à barres

  • bar - Comparaison de valeurs avec orientation horizontale
  • column - Comparaison de valeurs avec orientation verticale
↔ Les types bar et column sont interchangeables (orientation horizontale/verticale)
✓ Compatibles avec les options stacked et pivot
✓ Compatibles avec le mode continue

Graphiques linéaires

  • line - Évolution temporelle ou tendances
  • area - Évolution temporelle avec accent sur le volume
↔ Les types line et area sont interchangeables (avec/sans remplissage)
✓ Plusieurs styles de courbes disponibles
✓ Compatibles avec le mode continue

Graphiques matriciels

  • treemap - Données hiérarchiques avec proportions relatives
  • heatmap - Distribution d'intensité sur deux dimensions
↔ Les types treemap et heatmap sont interchangeables (visualisations alternatives)
✓ Supportent le groupement de données
⚠️ distributed et enableShades sont mutuellement exclusifs

Graphiques à intervalles

  • rangeBar - Intervalles de valeurs sous forme de barres
  • rangeArea - Intervalles de valeurs sous forme de zones
↔ Les types rangeArea, rangeBar et rangeColumn sont interchangeables
⚠️ Nécessitent des données min/max
✓ Compatible avec le mode horizontal pour rangeBar

Graphiques à entonnoir

  • funnel - Processus séquentiel ou conversion
  • pyramid - Hiérarchie de valeurs ordonnées
↔ Les types funnel et pyramid sont interchangeables
⚠️ Ne supportent pas le pivot
✓ Configuration spécifique pour le goulot

Comment choisir le bon type de graphique

  1. Nature des données : catégorielles, temporelles, proportionnelles, etc.
  2. Message à communiquer : comparaison, évolution, distribution, etc.
  3. Nombre de variables : une, deux ou plusieurs dimensions
  4. Compatibilité : transitions dynamiques entre types compatibles
