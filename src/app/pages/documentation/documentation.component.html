Installation & Démarrage

jQuery-Charts est une bibliothèque de visualisation de données pour Angular, qui simplifie l'utilisation de bibliothèques tierces comme ApexCharts ou Highcharts. Elle fournit une interface de programmation standardisée, ce qui signifie que vous utilisez toujours la même syntaxe Angular quel que soit le moteur de rendu choisi.

Concrètement, cela signifie que :

  • Vous n'avez pas besoin d'apprendre la syntaxe spécifique d'ApexCharts ou Highcharts
  • La configuration des graphiques reste identique quel que soit la bibliothèque sous-jacente
  • Vous pouvez créer rapidement des graphiques interactifs et personnalisables
  • La mise en place est simple et intuitive grâce à une syntaxe Angular native

1. Création du projet Angular

Si vous n'avez pas encore de projet Angular, créez-en un :

ng new my-chart-app
cd my-chart-app

2. Installation des dépendances

Installez les packages nécessaires :

npm install @oneteme/jquery-core @oneteme/jquery-charts

Si vous utilisez Apexcharts
npm install apexcharts

Si vous utilisez Highcharts
npm install highcharts

3. Configuration du module

Il existe deux façons d'importer le ChartComponent selon votre architecture :

Option A : Dans un module (app.module.ts)

import {{ '{' }} ChartComponent {{ '}' }} from '@oneteme/jquery-apexcharts';

@NgModule({{ '{' }}
  imports: [
    BrowserModule,
    ChartComponent
    ]
  {{ '}' }})

Option B : Dans un composant standalone

import {{ '{' }} ChartComponent {{ '}' }} from '@oneteme/jquery-apexcharts';

@Component({{ '{' }}
  standalone: true,
  imports: [ChartComponent]
// ... reste de la configuration du composant
{{ '}' }})

4. Utilisation dans un composant

Exemple d'utilisation simple :

<chart
  [type]="'pie'"
  [config]="config"
  [data]="data"
></chart>
