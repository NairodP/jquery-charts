Installation & Démarrage

1. Création du projet Angular

Si vous n'avez pas encore de projet Angular, créez-en un :

ng new my-chart-app
cd my-chart-app

2. Installation des dépendances

Installez les packages nécessaires :

npm install @oneteme/jquery-charts
npm install @oneteme/jquery-core

Si vous utilisez Apexcharts
npm install apexcharts

3. Configuration du module

Dans votre app.module.ts, importez ChartComponent :

import {{ '{' }} ChartComponent {{ '}' }} from '@oneteme/jquery-apexcharts';

@NgModule({{ '{' }}
  imports: [
    BrowserModule,
    ChartComponent
  ]
{{ '}' }})

4. Utilisation dans un composant

Exemple d'utilisation simple :

<chart 
  [type]="'pie'" 
  [config]="config" 
  [data]="data"
></chart>
